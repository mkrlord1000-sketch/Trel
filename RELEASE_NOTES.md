# Trel 0.2.0

Большой релиз: скины, локальные серверы и большая чистка кода.

## Что нового

### Скины
- Новая вкладка **Скин** в боковой панели.
- Можно загрузить свой PNG (64×64 или legacy 64×32) — drag&drop или выбор файла.
- 9 встроенных пресетов в футуристично-кибер-стиле: 3 мужских, 3 женских, 3 нейтральных.
- Скин виден **внутри игры** (1.6+) благодаря authlib-injector + локальному yggdrasil-mock-серверу. Лаунчер автоматически подкачивает агент при первом запуске.
- Поддержка моделей Steve (classic) и Alex (slim).
- На каталоге и в установленных версиях помечены те, где скин в игре не работает (rd, alpha, beta, 1.0–1.5).

### Локальные серверы
- Новая вкладка **Серверы**: создание/запуск/остановка локальных Minecraft-серверов прямо из лаунчера.
- Скачивание официального `server.jar` с серверов Mojang для любой релизной версии.
- Live-консоль с отправкой команд в stdin (op, say, stop, weather…).
- Настройки: MOTD, порт, max-players, gamemode, difficulty, PvP, whitelist.
- Авто-открытие порта в Брандмауэре Windows при первом запуске.
- Понятный блок «Как подключиться» с готовым адресом и кнопкой «Скопировать».

### Поддержка старых версий миров
- Видны и удаляются миры **Pre-Classic** (`level.dat`), **Indev** (`mclevel.dat`), **Infdev** (`saves/<имя>.mclevel`).

## Исправления

- **Безопасность:** все IPC-хендлеры теперь валидируют `versionId` / `worldName` / `serverId` — больше нет path traversal через renderer.
- **Forge на 1.21+:** перешли с NTFS junction для `mods/` на hard-link зеркало — Forge на JDK 21 больше не падает с `FileAlreadyExistsException`.
- **Скин в игре:** при смене скина URL текстуры теперь содержит SHA256 от PNG, поэтому Minecraft-кэш правильно инвалидируется. Дополнительно чистится `assets/skins/` чтобы старая текстура не подмешивалась.
- **Удаление миров:** rd-* / Indev / Infdev миры теперь корректно удаляются и не «воскресают» при следующем запуске старой версии.
- **Чужие миры:** лаунчер больше не показывает миры официального лаунчера Mojang из `%APPDATA%\.minecraft\saves` и не удаляет их при сбросе.
- **Счётчик версий:** отдельная папка ванили (родитель Forge) больше не считается отдельной установленной версией.
- **Кнопка «Играть»:** разблокируется сразу после старта игры, а не после её закрытия — можно запустить вторую сессию или другую версию.
- **Серверы:** `online-mode=false` и `enforce-secure-profile=false` по умолчанию + server-side authlib-injector — больше нет «Не удалось проверить имя пользователя».
- **Серверы:** на Windows используется `taskkill /T /F` для корректного освобождения порта.
- **Серверы:** перед запуском проверяется свободен ли порт — внятная ошибка вместо молчаливого падения.
- **Race conditions:** mutex для `accounts.json`, защита от двойного клика «Играть».
- **Утечки:** при закрытии окна корректно останавливается skin-сервер, очищается интервал manifest-проверок, гасятся серверы.
- **Атомарность:** flatten лоадер-профилей теперь устойчив к прерыванию посередине.
- **Сборка:** битые `accounts.json`/`settings.json` больше не валят лаунчер — мусорные записи фильтруются, типы валидируются.

## Технические детали

- Удалён мёртвый `LibraryPage.tsx` (~440 строк), мёртвый интерфейс `Instance`, неиспользуемые импорты.
- Локализованы дублирующиеся CSS-правила.
- В catch-all skin-сервера добавлено логирование неизвестных эндпоинтов.

## Как обновиться

Если вы уже использовали 0.1.0 — лаунчер обновится сам при следующем запуске. Иначе — скачайте `Trel-0.2.0-x64.exe` (NSIS-инсталлятор) или `Trel.exe` (portable) ниже.
